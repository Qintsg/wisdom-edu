# 演示数据导入说明

> 本文档说明如何导入适合答辩/演示的测试数据。

## 快速开始

```bash
cd backend

# 方式一：初始化数据库并创建基础测试数据，并自动导入课程资源 / Neo4j / GraphRAG
python tools.py pg-bootstrap

# 兼容下划线写法（与部分 TODO / 口头说明保持一致）
python tools.py pg_bootstrap

# 方式二：仅创建测试数据（不清空已有数据）
python tools.py create-test-data

# 如需单独重导完整课程资源（知识图谱、题库、视频、PPT 等）
python tools.py bootstrap-course-assets --course-name 大数据技术与应用

# 方式三：重建完整答辩演示环境（teacher / student / student1 / student2~5）
python tools.py rebuild-demo-data --course-name 大数据技术与应用
```

## 数据结构

### 所有学生共有

- 已加入"大数据技术与应用"课程班级
- 默认课程上下文设为大数据
- student1 额外加入所有班级

### `rebuild-demo-data` 额外保证

- 会额外创建答辩教师账号 `teacher` 与答辩学生账号 `student`
- 会创建答辩班级 `2024级大数据技术1班`
- `student2~5` 会被同步加入答辩班级，并强制清空该课程下的评测、画像、学习路径、作业与反馈轨迹
- `student2` 可直接用于“首次进入系统 → 初始评测入口”的新生演示链路
- `student` 用于“查看评测报告 → 学习路径 → 阶段测试反馈”的固定讲解链路

### student1 预置内容（`pg-bootstrap` / `pg_bootstrap` 基础态）

| 数据 | 说明 |
| ---- | ---- |
| 评测状态 | 能力评测、习惯问卷、知识评测均已完成 |
| 学习习惯 | 偏好视频学习、晚间学习、每周 5 天 |
| 初始评测 | 3 题完成，得分约 67%（大数据概述✓、Hadoop✗、Spark✓） |
| 知识掌握度 | 已答到的核心知识点按初测结果更新，其余大部分知识点保持 25% 左右保守基线 |
| 个人画像 | 已生成基础画像摘要、薄弱项和建议，可直接展示“刚完成初测”的系统反馈 |
| 学习路径 | 3 个节点：概念复盘(进行中)→Hadoop 入门(锁定)→Spark 预习(锁定) |
| 作业与反馈 | 仅保留初始评测反馈，不预置阶段作业与考试提交 |

> 如果需要 `student1` 具备“已完成评测 → 学习路径 → 阶段作业反馈”的完整讲解态，请使用 `rebuild-demo-data`，该命令会在基础态之上进一步补齐 `student / student1` 的完整轨迹。

### student2~5

仅完成注册和课程加入，无任何学习记录，适合演示"新学生首次进入"的流程；若需要固定一条答辩口径，优先使用 `student2`。

## 完整演示环境（答辩专用）

如需构建完整的答辩演示环境（包含更丰富的交互链路和预置反馈），使用：

```bash
python tools.py rebuild-demo-data --course-name 大数据技术与应用
```

该命令会：

1. 清空并重建数据库
2. 创建答辩专用账号（`teacher / student / student1 / student2~5`）
3. 导入完整课程资源，并同步 Neo4j / GraphRAG
4. 为 `student / student1` 预置评测结果、学习路径、阶段测试反馈等
5. 为 `student2~5` 保持“仅入班无轨迹”的新生状态
6. 保证正式答辩链路可直接按讲稿演示

## 推荐答辩链路账号

| 场景 | 推荐账号 | 说明 |
| ---- | -------- | ---- |
| 教师端建课 / 题库编辑 | `teacher` | 绑定答辩班级与主课程 |
| 新生首次进入 | `student2` | 无测评、无画像、无路径 |
| 固定完整讲解 | `student` | 已预置评测报告、学习路径、阶段测试反馈 |
| 备用完整链路 | `student1` | 同样具备完整学习数据，可用于容灾切换 |

## 数据源

测试数据配置文件位于 `backend/tools/testdata.json5`，支持 JSON5 格式（带注释和尾随逗号）。修改该文件后重新运行 `create-test-data` 即可生效。
